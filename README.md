LiveNotes is a collaborative platform that allows users to share and edit notes in real-time with friends and colleagues. Whether you're brainstorming ideas, managing tasks, or sharing quick notes, LiveNotes makes collaboration seamless and efficient.

## 🌟 Features
- 🕒 Real-Time Collaboration – Edit and share notes instantly with others
- 🔒 Secure Sharing – End-to-end encryption for your notes
- 📋 Organized Notes – Categorize and tag notes for easy access
- 🌐 Cross-Platform – Access your notes from any device
- 🎨 Customizable Themes – Personalize your workspace
- 📊 Activity Tracking – Monitor changes and contributions
- 🛠️ Developer-Friendly API – Integrate LiveNotes into your apps

## 🛠️ Technologies Used
- Next.js – Frontend framework
- TailwindCSS – Styling and responsive UI
- Node.js – Backend server
- Express.js – API handling
- MongoDB – Database management
- WebSockets – Real-time communication


## 🚀 How to Use

- 📝 Create an account or log in.
- 📂 Create a new note or open an existing one.
- 👥 Share the note link with collaborators.
- ✏️ Edit and collaborate in real-time.
- 🔖 Organize notes with tags and categories.
- 📊 Monitor changes and contributions.
